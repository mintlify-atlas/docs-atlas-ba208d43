## Overview

Elemental Battlecards offers two distinct paths to victory, allowing for diverse strategic approaches. You can win by either collecting all card types or gathering enough essence tokens.

## Victory Conditions

</Card>

<Note>
Card level doesn't matter for this condition - you just need at least one card of each type somewhere in your possession (hand, field, or deck).
</Note>

### Condition 2: Essence Collection

<Card title="Collect 6 Essence Tokens" icon="gem">
  Be the first player to collect 6 essence tokens to win the game.
</Card>

```javascript
// From Frontend/src/helpers/constants.js

export const GAME_CONFIG = {
    ESSENCES_TO_WIN: 6,  // Need 6 essence tokens
};
```

**How to earn essence:**

Essence tokens are awarded when you defeat enemy cards in combat. The essence system is managed by the `EssenceManager` class:

```javascript
// From Frontend/src/game_objects/player.js

import EssenceManager from './essences.js';

export default class Player {
    constructor(id, scene) {
        this.id = id;
        this.scene = scene;
        this.essences = new EssenceManager();  // Tracks essence tokens
        // ... other player properties
    }
}
```

<Tabs>
  <Tab title="Earning Essences">

    The number of essences earned may depend on:
    - The level of the defeated card
    - The type of combat victory
    - Special game events
    
    <Note>
    Check the EssenceManager implementation for exact essence award rules.
    </Note>
  </Tab>
  <Tab title="Tracking Progress">
    **Monitor your essences**: Your essence count is tracked throughout the game.
    
    Once you reach 6 essences, victory is automatically triggered.
    
    **Strategic tip**: Focus on winning combats to rapidly accumulate essences.
  </Tab>
</Tabs>

## Strategic Considerations

### Dual-Path Strategy

<Note>
The best players pursue both victory conditions simultaneously, adapting based on game flow.
</Note>

**Example strategic decisions:**

```
Situation: You have 5/6 types and 3 essences
Decision: Focus on drawing cards to find the missing type
Action: Play defensively, draw aggressively

Situation: You have 3/6 types but 5 essences
Decision: Go for the essence victory
Action: Attack aggressively to get the final essence

Situation: You have 4/6 types and 4 essences
Decision: Stay flexible and pressure opponent
Action: Maintain board control, force opponent errors
```

## Game Configuration

All victory-related constants are defined in the game configuration:

```javascript
// From Frontend/src/helpers/constants.js

export const GAME_CONFIG = {
    MAX_HAND_SIZE: 4,            // Limits hand size
    MAX_FIELD_SIZE: 6,           // Limits field size
    INITIAL_DECK_SIZE: 48,       // Starting deck
    TURN_TIME_LIMIT_MS: 12000,   // 12 seconds per turn
    MANDATORY_ATTACK_TURN: 3,    // Must attack by turn 3
    UNIQUE_TYPES_TO_WIN: 6,      // Victory condition 1
    ESSENCES_TO_WIN: 6,          // Victory condition 2
};
```

<Warning>
The mandatory attack rule on turn 3 prevents pure defensive strategies. You must engage in combat, which opens paths for both victory conditions.
</Warning>

## Victory Triggers

The game continuously checks for victory conditions:

- **After each draw**: Check if player has all 6 unique types
- **After each combat**: Check if player has reached 6 essences
- **After each turn**: Validate victory conditions

The first player to meet either condition wins immediately.

## Advanced Tips

<Tabs>
  <Tab title="Tempo Control">
    Control the game pace based on your victory path:
    
    - **Leading in types?** Slow down, play defensively
    - **Leading in essences?** Speed up, force combat
    - **Behind in both?** Disrupt opponent's strategy
  </Tab>
  <Tab title="Opponent Denial">
    Recognize your opponent's path and counter it:
    
    - If they're close to 6 types: Force them to lose card diversity
    - If they're close to 6 essences: Play defensively, avoid feeding them wins
    - If they're threatening both: Focus on your own fastest path
  </Tab>
  <Tab title="Resource Management">
    Balance deck, hand, and field:
    
    - **Deck**: Contains all types eventually
    - **Hand**: Limited to 4 cards
    - **Field**: Limited to 6 spaces
    - **Graveyard**: Recycles back to deck
    
    Manage these zones to maximize your victory chances.
  </Tab>
</Tabs>
